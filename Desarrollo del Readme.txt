# web3_bank

## Descripción
web3_bank es una aplicación bancaria descentralizada que utiliza tecnología web3 para ofrecer servicios financieros innovadores y seguros. Este proyecto permite a los usuarios gestionar sus activos digitales de manera eficiente y transparente, aprovechando las ventajas de la blockchain y los contratos inteligentes.

## Características
- **Gestión de Activos**: Permite a los usuarios almacenar y gestionar sus criptomonedas y tokens.
- **Transacciones Seguras**: Realiza transacciones rápidas y seguras a través de la tecnología blockchain.
- **Interfaz de Usuario Intuitiva**: Proporciona una experiencia de usuario amigable para facilitar la navegación y el uso.

## Tecnologías Utilizadas
- **JavaScript**: Para la lógica del lado del cliente.
- **Solidity**: Para el desarrollo de contratos inteligentes en la blockchain.
- **HTML/CSS**: Para la estructura y el estilo de la aplicación.
